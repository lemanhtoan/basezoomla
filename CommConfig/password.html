Maintaining Your PDG Commerce Administration Password


The new version of PDG Commerce that you have recently installed allows you to set up multiple user accounts with restricted access for your administration panel. Changing passwords, adding new users, modifying user privilages and other similar tasks are now accomplished within the administration panel's "Users" section.

Since older versions of the administration panel required only a password, we have set up your new version to automatically assign your existing password to the user account "Admin" (Note: both user names and passwords are case-sensitive).

To log into your administration panel, please select "Administration" from the menu to the left.


Note concerning your administration panel password:

New versions of PDG Commerce require that your password be a minimum of 8 characters and must include either a number or a special character (e.g. $, #, %, etc.). PDG Software advises that you change your passwords on a regular basis.
