Upgrade Information
You are using Commerce 2003 executables, and your configuration indicates that you have not upgraded your site.

You must upgrade your site configuration in order for Commerce 2003 to operate properly.

Due to the amount of time it takes to convert a site to be Commerce 2003 compatible, there is no Internet upgrade path. You must use the PDG Import/Export/Upgrade Utility. This tool is only available for Windows Operating Systems, but it can connect and upgrade any Commerce or Cart 2002 web site regardless of the web server's OS.
