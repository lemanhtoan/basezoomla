General user name and password information
User name cannot contain these characters: !@#$%%^&*()_+=|/<>,.-\:;`~
User name cannot be test or Admin.
User name must be at least 5 characters long.

Password must be at least 8 characters long.
Password must contain a number or special character.
Password must contain upper and lower case letters.

Email address must follow email naming conventions (ex. someone@somesite.zone)

Secret phrase must be at least 16 characters long.
