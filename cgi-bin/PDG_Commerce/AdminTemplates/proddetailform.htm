    Customer Category Pricing

    This section allows you to enter product specific discounts for a customer category. Enter the discount amount in the input boxes provided. Entries left blank will default to the standard price or the default category price for the product. To set a new price enter the discount amount (for taxable price, enter the taxable discount immediately following the unit discount, separating the two with a colon. e.g. 12.00:10.00) then click the 'Submit Changes' button

    Product Options

    This section defines the available options (eg. Color) for this product. Each option has definable selections (eg. Red, White, and Blue). When customers add this product to their shopping basket, they will choose one selection for each option. To remove a particular option, press the Remove button for that option. To add an option, enter its description and press Add. To view and modify option selections for a specific option, press the Details button next to that option.

    User Defined Data

    These fields can be customized by the Administrator to add extra information about each product. Each user defined field should represent the same thing for all products. For example field one can be used for upsales. You can enter links for other products to buy, and then place the tag for User defined one on the Item page. The tag will place the contents of User defined one on the Item page. This is only one example of how User Defined fields can be used.
