USPS User ID and Password

In Order to use U.S. Postal Service on-line tools, you need to register at www.uspswebtools.com. If you have already registered, you still need to call USPS and make your account live. Once your USPS account is made live, you can type in your user ID and password in the space provided below and click on 'Save USPS User Information'. After your USPS information is saved, you can set up shipping methods using USPS Services.
