View, Delete and Print Shipping Labels
Below is a listing of all the labels currently stored in your shiplabel directory. If you would like to view a label click the 'Display' link next to the order. This will open a new window with printing instructions and the label. If you would like to delete a label click the 'Delete' link next to the order you would like to delete. This will permanently delete the shipping label. If you would like to delete all your labels (this will permanently remove all your shipping labels) click the following link: Remove All Shipping Labels
